# set-trek
Starship game built with custom DirectX11 3D graphics engine. This was a school project that I took to the next level by incorporating 3D rendering into a 2D scene.

### How to run
1. Open folder
2. Click on main.exe

### Controls
Click - Move

E - Warp to planet

1 - Basic missile

2 - Fast missile

3 - Slow, heavy missile

4 - Heal
